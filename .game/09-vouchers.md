# Vouchers

Permanent, run-wide upgrades bought from the shop; one Voucher appears per Ante.

> As of: v1.0.1 (base game, ~1.0.1n/1.0.1o)
> Total: 32 vouchers (16 base + 16 upgrade pairs)

## How vouchers work

- **One per Ante**: Exactly one Voucher is offered in the shop each Ante. It does **not** refresh when you reroll the shop.
- **Permanent**: A purchased Voucher applies for the rest of the run. Each Voucher can be bought only once per run.
- **Cost**: Every Voucher costs **$10** (both base and upgrade tiers).
- **Two tiers**: Each base Voucher has an upgraded "tier 2" version. The upgrade can only appear after its base Voucher has been purchased in the same run, **and** the upgrade must be unlocked (via its unlock condition) to appear at all. Of the 32 vouchers, 16 base vouchers are available from the start; the 16 upgrades require unlocking.
- Effects of base + upgrade generally **stack** (e.g., +1 then +1 more shop slot).

## Notes & nuances

- **Interest mechanic**: Standard interest is $1 per $5 held, capped at $5/round (max $25 held). Seed Money raises the cap to $10/round; Money Tree to $20/round.
- **Hieroglyph trade-off**: Lowering Ante reduces the score threshold curve but costs a hand each round — a "skip ahead" tool, not pure upside. Petroglyph stacks another -1 Ante at the cost of a discard.
- **Antimatter** the *voucher* grants a Joker slot; this is distinct from the **Antimatter** Joker *edition* (Negative) which also grants a Joker slot — different objects, same name family.
- **Observatory** only buffs Planet cards while they are held in the consumable area (not consumed); it multiplies that hand's Mult by X1.5 (stacking multiplicatively per copy held).
- **Telescope** guarantees the most-played hand's Planet in Celestial Packs, enabling reliable leveling of a single hand.
- **Omen Globe** lets Spectral cards roll inside Arcana Packs (it does not add a slot — Crystal Ball, its base, is the slot grant).
