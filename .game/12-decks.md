# Decks

The 15 starting decks in Balatro. Each deck applies a permanent run-wide modifier and is selected before a run begins.

> As of: v1.0.1 (base game, ~1.0.1n/1.0.1o). Confirmed count: 15 decks.

Decks are unlocked in a roughly linear chain: discovering items in the collection unlocks the first batch of colored decks, then winning a run with each colored deck unlocks a "themed" deck, then clearing successive Stakes unlocks the remaining advanced decks. The Red Deck is available from the start.

## All 15 Decks

| Deck | Effect / Modifier | Unlock requirement |
|------|-------------------|--------------------|
| Red Deck | +1 discard every round | Available from the start |
| Blue Deck | +1 hand every round | Discover at least 20 items in your collection |
| Yellow Deck | Start the run with an extra $10 | Discover at least 50 items in your collection |
| Green Deck | At end of each Round: earn $2 per remaining Hand and $1 per remaining Discard; earn **no interest** | Discover at least 75 items in your collection |
| Black Deck | +1 Joker slot; **-1 hand** every round | Discover at least 100 items in your collection |
| Magic Deck | Start run with the **Crystal Ball** voucher and 2 copies of **The Fool** | Win a run with the Red Deck (any stake) |
| Nebula Deck | Start run with the **Telescope** voucher; **-1 consumable slot** | Win a run with the Blue Deck (any stake) |
| Ghost Deck | **Spectral** cards may appear in the shop; start with a **Hex** card | Win a run with the Yellow Deck (any stake) |
| Abandoned Deck | Start run with **no Face Cards** (no Kings, Queens, or Jacks) in the deck | Win a run with the Green Deck (any stake) |
| Checkered Deck | Start run with **26 Spades and 26 Hearts** (no Clubs or Diamonds); still 52 cards | Win a run with the Black Deck (any stake) |
| Zodiac Deck | Start run with the **Tarot Merchant**, **Planet Merchant**, and **Overstock** vouchers | Win a run on the **Red Stake** or higher (any deck) |
| Painted Deck | +2 hand size; **-1 Joker slot** | Win a run on the **Green Stake** or higher (any deck) |
| Anaglyph Deck | After defeating each **Boss Blind**, gain a **Double Tag** | Win a run on the **Black Stake** or higher (any deck) |
| Plasma Deck | **Balance** Chips and Mult when calculating the score for the played hand; **X2 base Blind size** | Win a run on the **Blue Stake** or higher (any deck) |
| Erratic Deck | All **Ranks and Suits** of the 52 cards are randomized | Win a run on the **Orange Stake** or higher (any deck) |

## Notes

- **Green Deck** replaces the normal interest mechanic with a flat money payout based on unused Hands/Discards. It earns money for *playing efficiently* rather than for hoarding cash, so it interacts very differently with economy strategies.
- **Black Deck** is a net trade of one card-play resource (hand) for one Joker slot — strong for scaling builds, weak for consistency.
- **Magic Deck**: Crystal Ball adds +1 consumable slot; the 2 copies of The Fool let it copy the last-used Tarot/Planet early.
- **Nebula Deck**: Telescope makes the most-played poker hand's Planet card guaranteed in Celestial Packs; the trade-off is one fewer consumable slot.
- **Ghost Deck**: starting Hex card immediately applies a Polychrome to a random Joker (and destroys the others), so the deck typically opens with a strong Joker.
- **Abandoned Deck** removes 12 face cards, leaving a 40-card deck (Ace through 10). Improves consistency for several hand types and pairs well with effects that care about non-face cards.
- **Checkered Deck** guarantees Flush-heavy and same-suit synergies (only 2 suits present).
- **Plasma Deck**: "Balance" averages Chips and Mult to (Chips + Mult) / 2 each before multiplying, which massively boosts hands where Chips and Mult are lopsided; X2 base Blind size makes every blind's required score double.
- **Erratic Deck** is the highest-variance deck: rank and suit of every card is rolled at run start, so duplicates and missing ranks/suits are common.
- **Anaglyph Deck**: the Double Tag granted after each Boss Blind itself duplicates the next purchased Tag, enabling Tag-stacking economy/Joker strategies.

⚠️ Exact verbatim in-game tooltip wording can vary slightly by patch; effects above are cross-checked across the community wiki and multiple guides and reflect base-game v1.0.x behavior.
